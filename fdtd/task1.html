Task 1: Absorbing Boundary Conditions

For the 1-dimensional wave equation given by \begin{equation} \dfrac{\partial^2 U}{\partial x^2} = \dfrac{1}{c^2} \dfrac{\partial^2 U}{\partial t^2} \end{equation} we can define an operator $G$ where \begin{equation} G = \dfrac{\partial^2}{\partial x^2} - \dfrac{1}{c^2} \dfrac{\partial^2}{\partial t^2} \end{equation} so that the wave equation can be written easily as \begin{equation} GU = 0 \end{equation} We can separate $G$ such that \begin{equation} G = G^+ G^- \end{equation} \begin{equation} G^+ = \dfrac{\partial}{\partial x} + \dfrac{1}{c} \dfrac{\partial}{\partial t} \end{equation} \begin{equation} G^- = \dfrac{\partial}{\partial x} - \dfrac{1}{c} \dfrac{\partial}{\partial t} \end{equation} and it was previously shown that at the left side of the boundary, where $x=0$, \begin{equation} G^-U = 0 \qquad \longrightarrow \qquad \dfrac{\partial U}{\partial x} - \dfrac{1}{c} \dfrac{\partial U}{\partial t} = 0 \end{equation} and at the right side, at $x=d$, \begin{equation} G^+U = 0 \qquad \longrightarrow \qquad \dfrac{\partial U}{\partial x} + \dfrac{1}{c} \dfrac{\partial U}{\partial t} = 0 \end{equation}

Expressing the first differential equation as a discretized difference, this \begin{equation} \dfrac{\partial E}{\partial x} = \dfrac{1}{c} \dfrac{\partial E}{\partial t} \end{equation} becomes \begin{equation} \dfrac{1}{\Delta t} \left ( E \Big |{1/2}^{n+1} - E \Big |{1/2}^{n} \right ) = \dfrac{1}{c \Delta x} \left ( E \Big |{1}^{n+1/2} - E \Big |{0}^{n+1/2} \right ) \end{equation} expressing the terms involving 1/2 into averages, \begin{equation} \dfrac{1}{2 \Delta t} \left [ \left ( E \Big |{0}^{n+1} + E \Big |_1^{n+1} \right ) - \left( E \Big |{1}^{n} + E \Big |{0}^{n} \right ) \right ] = \dfrac{1}{2c \Delta x} \left [ \left ( E \Big |{1}^{n+1} + E \Big |{1}^{n} \right ) - \left ( E \Big |{0}^{n+1} + E \Big |{0}^{n} \right ) \right ] \end{equation} Solving for $E \Big |{0}^{n+1}$ \begin{equation} E \Big |{0}^{n+1} = E \Big |{1}^{n} - \left ( \dfrac{c \Delta x - \Delta t}{c \Delta x + \Delta t} \right ) \left ( E \Big |{0}^{n} - E \Big |{1}^{n+1} \right ) \end{equation}

For the other side, \begin{equation} E \Big |{i}^{n+1} = E \Big |{i-1}^{n} - \left ( \dfrac{c \Delta x - \Delta t}{c \Delta x + \Delta t} \right ) \left ( E \Big |{i}^{n} - E \Big |{i-1}^{n+1} \right ) \end{equation}

%matplotlib inline
from matplotlib import pyplot as plt
#from matplotlib import animation

import numpy as np
import math as m

nx = 300

imp0 = 337.0 #impedance

#initialize the fields at zero
ez = np.zeros(nx)
hy = np.zeros(nx)
x = np.arange(0,nx-1,1)

epsilon = 1
#eps= np.ones(nx)
#eps[:] = epsilon

#Source 
srcori = int(nx/2) #source is at center
srcwid = 30.0*np.sqrt(epsilon) #source width
srcdel = 10*srcwid #source delay
nt = int(3*(nx+srcdel)*np.sqrt(epsilon))

c = 1/np.sqrt(epsilon)
# here, we have dx and dt be both equal to one
# based on Inan p.177
a = (c-1)/(c+1)

# Left boundary (set to zero because everything is initially zero)
enp10, en1, en0, enp11 = 0,0,0,0
hnp10, hn1, hn0, hnp11 = 0,0,0,0
# Right boundary
enp1i, enim1, eni, enp1im1 = 0,0,0,0
hnp1i, hnim1, hni, hnp1im1 = 0,0,0,0

fignum = 0
for dt in range(0,nt):
    ######################
    #Magnetic field
    ######################
    hy[x] = hy[x] + (ez[x+1] - ez[x])/imp0
    #abc at left
    hn1 = hy[1]
    hnp10 = hn1 - a*(hn0 - hnp11)
    hy[0] = hnp10
    hnp11 = hn1
    hnp10 = hn0
    #abc at right
    hnim1 = hy[-2]
    hnp1i = hnim1 - a*(hni - hnp1im1)
    hy[-1] = hnp1i
    hnim1 = hnp1im1
    hni = hnp1i 
#    wrm1_np1 = hy[-2]
#    wr_np1 = -wrm1_nm1 + a*(wrm1_np1+wr_nm1) + b*(wr_n+wrm1_n)
#    hy[-1] = wr_np1
    #Cycle field values at boundary
#    wr_nm1, wrm1_nm1 = wr_n, wrm1_n
#    wr_n, wrm1_n = wr_np1, wrm1_np1
    ######################
    #Electric field
    ######################
#    ez[x+1] = ez[x+1] + (hy[x+1]-hy[x])*imp0/eps[x+1]
    ez[x+1] = ez[x+1] + (hy[x+1]-hy[x])*imp0/epsilon
    ez[srcori] += m.exp(-((dt-srcdel)*(dt-srcdel))/(srcwid*srcwid))
    #abc at left
    en1 = ez[1]
    enp10 = en1 - a*(en0 - enp11)
    ez[0] = enp10
    enp11 = en1
    enp10 = en0
    #abc at right
    enim1 = ez[-2]
    enp1i = enim1 - a*(eni - enp1im1)
    ez[-1] = enp1i
    enim1 = enp1im1
    eni = enp1i
#    wlp1_np1 = ez[1]
#    wl_np1 = -wlp1_nm1 + a*(wlp1_np1+wl_nm1) + b*(wl_n+wlp1_n)
#    ez[0] = wl_np1
    #Cycle field values at boundary
#    wl_nm1, wlp1_nm1 = wl_n, wlp1_n
#    wl_n, wlp1_n = wl_np1, wlp1_np1

    plt.hold(True)
    if (dt % 60 == 0 or dt == nt-2):
        fignum = fignum + 1
        plt.figure(fignum)
        plt.xlabel("Position")
        plt.ylabel("E-field Amplitude")
        plt.title("E-field at t = "+ str(dt))
        plt.plot(ez)
        
        if (dt != nt-2):
            plt.plot(hy*imp0)
